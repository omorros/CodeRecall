# CodeRecall

A RAG (Retrieval-Augmented Generation) application that lets you chat with GitHub repositories. Built to learn AI engineering fundamentals: embeddings, vector databases, and LLM context augmentation.

## Tech Stack

- **Frontend:** Next.js (App Router), TypeScript, Tailwind CSS
- **Backend:** Python 3.11+, FastAPI
- **Database:** PostgreSQL 16 + pgvector
- **Queue:** Redis + RQ
- **AI:** OpenAI API (gpt-4o-mini for chat, text-embedding-3-small for embeddings)
